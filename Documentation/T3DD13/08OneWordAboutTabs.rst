==========================================
One word about tabs
==========================================

Indentation is easy if you are using blanks. It is possible
to use tabs as well. But tabs make it difficult to give
the ReST source a nice layout. It's even possible to mix
blanks and tabs - but you'll definitely run into trouble.

So you *may* decide to listen to my advice about tabs:

.. admonition:: One word about tabs:

      Don't use them!
